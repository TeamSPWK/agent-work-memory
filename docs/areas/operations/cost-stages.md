- Stack: Clerk + Supabase Tokyo + Fly.io NRT + Cloudflare DNS·CDN + 토스페이먼츠 + 자동 세금계산서
- 가격: PRD §7.2 — Starter 10만 / Team 25만 / Pro 50만 (월), 연결제 25% 할인 별도
- 첫 결제 시점: PRD §10.1 D1 단계 (디자인 파트너 5팀 50% 할인)

본 표는 *가설*. 실제 청구서·MRR로 매달 갱신 (`mrr-tracker.md`·실제 청구서 스크린샷 누적).

## 비용 함정

1. **Supabase egress $0.09/GB** — AI 캡처 이벤트 외부 다운로드 시 청구서 폭증. *spend cap 기본 ON 확인 필수*.
2. **Clerk B2B 무제한 멤버 한도** — Pro $25는 *조직 멤버 20명까지* 무료. 50명 팀 첫 등장 시 +$100/mo 애드온.
3. **Fly.io Free tier 2026-01 종료** — 이전 무료 항목 유료화. always-on 머신은 무조건 과금. 안 쓸 때 stop/start 직접 관리.
4. **토스페이먼츠 PG 수수료** — 가맹 심사·요율 협상에 따라 2.5~3.3%. 위 표는 2.9% 가정. 첫 가맹 시 정확 요율 확인.
5. **세금계산서 자동화 도구 가격** — 아임포트/Bizmark 정확 가격은 *발행 건수별 견적*. 월 5~10만원 가정했지만 실제는 다를 수 있음.

## 추정의 한계

- Clerk B2B 정확 가격: 멤버 수 분포에 따라 변동. 위 표는 *평균 15명/팀* 가정
- Supabase 컴퓨트: 워크로드(이벤트 insert 빈도)에 따라 변동
- 토스페이먼츠 PG·세금계산서 자동화: *공시되지 않은 견적 영역*
- 본 산정은 PRD §7 가격 가설(Team 25만원/15 OP)이 맞다고 가정. PRD §11.6 Open Question 8번대로 가격 자체가 틀리면 매출 추정도 함께 변동
