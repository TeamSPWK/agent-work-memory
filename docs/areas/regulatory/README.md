## 운영 원칙

- 분기 1회 새 가이드라인·시행령 점검 (Weekly Review에서 알림)
- 변경 시 PRD §11 Risks 또는 §1 외부 동향 갱신
- 외부 자문 사용 시 결과를 본 영역에 누적 (실명 자문 정보는 git-crypt)

- 인공지능기본법 *2026-01-22 시행*은 PRD §1 신호 7번 + §2.3 트리거 A (★★★)의 직접 근거
- 시행 1개월 전부터 *결제 의지 가속*이 가설 — Weekly Review에서 추적
