# Claude Design — v0.2 외부 페이지 라운드 프롬프트 (Round 3)

> 미가입자가 도달하는 *public-facing* 페이지 시안. v0(H1·H2·H3 16화면) + v0.1(H4·Workspace·Settings 7화면) 의 *inside-app* 23화면 lock 후, PRD v2.1 §12에서 빠진 영역으로 정의된 외부 페이지를 채우는 라운드.
> v0.1과 동일 토큰·anonymization·1인 운영 카피 룰 유지.

---

## 사용 방법

1. claude.ai/design에서 **기존 v0.1 세션을 이어서 작업** (v0 → v0.1 → v0.2 누적)
2. 본 문서의 **Master Prompt v0.2** 섹션 복붙
3. 결과 번들을 `docs/projects/plans/p0-design-v0.2/`로 저장 후 NOVA-STATE / m2.5 Plan 진입
