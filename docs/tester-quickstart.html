테스터 Quick Start

5분 안에 AWM 작동 확인하기

3 단계로 끝납니다. 명령 한 줄 실행 → Claude Code 재시작 → 브라우저에서 확인.

1초기화 한 줄~2분

레포 클론 후 루트 디렉토리에서:

git clone https://github.com/TeamSPWK/agent-work-memory.git
cd agent-work-memory
npm run init

다섯 단계가 자동으로 실행됩니다 — Node 버전 확인, 의존성 설치, 빌드, Claude Code hook 자동 머지, 로컬 서버 백그라운드 기동.

요구사항 · Node 20 이상, macOS 또는 Linux. Claude Code(claude)는 미리 설치되어 있어야 합니다.

2Claude Code 재시작~30초

현재 열려 있는 Claude Code 창을 모두 종료하고 다시 실행합니다. 캡처 hook은 Claude Code 시작 시점에만 로드되므로 재시작이 필수입니다.

왜 재시작이 필요한가요? Claude Code는 .claude/settings.local.json을 시작 시 한 번 읽습니다. 실행 중에는 변경을 감지하지 않습니다 (Anthropic 측 제한). 1회 재시작 후로는 다시 안 해도 됩니다.

3첫 세션 + 확인~2분

새 Claude Code 세션에서 평소처럼 작업을 시작합니다 (아무 prompt). 몇 초 후 터미널에서:

node bin/awm.mjs audit show --last 3

최근 3개 이벤트의 prev·hash가 보이면 캡처 작동 OK. 브라우저에서:

http://127.0.0.1:5173/today

본인 세션이 Today 화면 우측 카드에 나타나면 완료입니다.

  • npm run serve:status — PID + 포트 LISTEN 확인
  • node bin/awm.mjs audit verify — chain 무결성 (exit 0 = OK)
  • npm run serve:logs — 서버 로그 마지막 40줄

자주 묻는 질문

내 작업 데이터는 어디에 저장되나요?

모두 로컬 디스크 — 프로젝트 루트의 .awm/ 디렉토리입니다. 외부 서버 전송은 없습니다 (V0 자체 캡처 단계).

저장되는 것: 요약·이벤트·메타데이터·해시·링크. 저장되지 않는 것: raw transcript 본문, 명령 stdout 전체.

hook이 다른 프로젝트에도 영향을 주나요?

아니요. .claude/settings.local.json은 이 디렉토리의 Claude Code 세션에만 적용됩니다. 다른 프로젝트에서 Claude Code를 켜면 그곳의 settings를 사용합니다.

서버를 종료하려면? 
npm run serve:stop

PID 파일로 깔끔히 종료됩니다. hook은 그대로 살아있어서 Claude Code 세션은 계속 캡처합니다 (다음에 npm run serve 하면 그동안 캡처된 이벤트도 같이 보입니다).

완전히 제거하려면? 
npm run serve:stop
rm -rf .awm dist node_modules
rm .claude/settings.local.json

Claude Code를 다시 재시작하면 hook이 사라집니다.

문제 해결

Node 버전이 낮다고 나옵니다

Node 20 이상이 필요합니다. nodejs.org 또는 brew install node (macOS).

포트 5173이 이미 사용 중입니다 
AWM_PORT=5174 npm run serve:restart

다른 포트로 재기동 가능합니다.

audit show 출력이 비어 있습니다

Claude Code를 재시작했는지 확인하세요. 재시작했는데도 비어있으면:

cat .claude/settings.local.json

이 파일에 hooks.SessionStart 등이 보여야 합니다. 없으면 다시:

node bin/awm.mjs capture install claude --auto-merge
브라우저에서 화면이 안 열립니다 
npm run serve:status
npm run serve:logs

서버가 LISTEN 중인지, 로그에 에러가 없는지 확인. 필요시 npm run serve:restart.
